Document Management System

MuK IT GmbH - www.mukit.at

Overview

MuK Documents is a module to create, manage and view files directly within Odoo. This module is only the basis for an entire ecosystem of apps that extend and seamlessly integrate with the document management system.

Views

The standard Odoo views, Kanban, List and Form are implemented for folders and Files. The Kanban View can be used to directly display the subfolders or files of a particular folder.

Document Settings

There is the possibility to create own settings for different folder structure cures. This gives you the option of adapting the structure even further to your requirements.

Documents Extension

This module is just the basis for a complete ecosystem of applications that can expand and seamlessly integrate into the document management system. You can download the extensions in the store.
