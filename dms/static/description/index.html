MuK Documents

Overview

MuK Documents is a module to create, manage and view files directly within Odoo. By installing module extensions additional features can be enabled.

✔ Access Control (MuK Documents Access)
✔ File Store (MuK Documents File)
✔ More... (MuK Apps)

Documents View

All in one view to easily manage all your documents. Provides a context menu and a preview to speed up your work.

Documents Settings

It is possible to have multiple settings to customize your document management system.
